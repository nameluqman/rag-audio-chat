# RAG Audio Chat Frontend

React frontend for the RAG Audio Chat system.

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

The frontend will start on `http://localhost:3000` and connect to the backend at `http://localhost:5000`.

## Features

- Text-based Q&A interface
- Audio recording (optional)
- Real-time chat experience
- Responsive design
